Passo 3 - App web + PGap + Android (11 de 12)

Exportar a aplicação para apk, para envio para o google play (store)

  • No eclipse, abrir o menu "File" -> "Export"
  • No ecrã que abrir, escolher a opção "Export Android application" e activar "next"
  • Escolher o projecto e activar "next"
  • Escolher a keystore "z_step3.keystore", password: hybrid_app
  • Escolher a chave existente "hybrid_app", password: hybrid_app
  • Escolher um ficheiro para guardar o apk, ex: HybridApp.apk
  • Carregar no "finish"

Passo 3 - App web + PGap + Android (12 de 12)

Instalação do apk no android smartphone

  • Ligar o smartphone por usb ao computador
  • Na linha de comandos executar: adb install HybridApp.apk
